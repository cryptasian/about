Student Loan Repayment Chatbot Demo

An initial concept for a student loan repayment help bot that will be able to help a user find out who their loan servicer is, what payment methods are available and prequalify the user for a forbearance, deferment and/or an income driven repayment plan based on the responses to a set of standard questions.

Questions mimic the same question flow that a student loan servicer representative would ask over the phone, but the user is able to interact with the chat bot on their own time.
